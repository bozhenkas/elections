# Elections СС СПбГУТ

## Стек
- React 18 + Vite
- CSS Modules (без UI-библиотек)
- Recharts для графиков результатов

## Дизайн-система
- Фон: #0F0F0F
- Блоки: #243434
- Акцент: #CDDCDC
- Розовый (микро): #FFCECE — максимум 2-3 элемента на экран
- Шрифт display: Serpantin (uppercase заголовки)
- Шрифт декор: Decor Italic (подписи, слово «выборы»)
- Шрифт UI: системный гротеск (Inter / SF Pro)

## Архитектура
- Single-page, без роутинга
- Переход к выборам — модальное окно (не отдельный роут)
- Все API-вызовы в папке src/api/ с заглушками
- Каждая заглушка помечена TODO для бэкенд-разработчика

## Правила
- Не использовать тени, градиенты на фоне
- Бордеры 0.5px #CDDCDC
- Скругления 12-16px для карточек
- Serpantin ТОЛЬКО для крупных заголовков
- Decor Italic ТОЛЬКО для мелких подписей-акцентов
- Розовый FFCECE — только бейджи статуса
