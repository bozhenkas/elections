# ТЗ для дизайнера — Elections СС СПбГУТ

## 1. Общая информация

Сайт выборов Студенческого совета СПбГУТ. Single-page приложение, все детали выборов открываются в модальных окнах.

### Дизайн-система

| Токен | Значение | Использование |
|-------|----------|---------------|
| `--bg` | `#0F0F0F` | Фон страницы |
| `--surface` | `#243434` | Фон блоков, карточек, модалок |
| `--accent` | `#CDDCDC` | Текст заголовков, бордеры, акценты |
| `--pink` | `#FFCECE` | Только бейджи статуса (макс. 2-3 на экран) |
| `--white` | `#FFFFFF` | Основной текст |
| `--muted` | `rgba(205, 220, 220, 0.6)` | Вторичный текст |
| `--border` | `#CDDCDC` | Бордеры (0.5px) |

### Типографика

| Шрифт | Назначение | Правила |
|-------|------------|---------|
| **Serpantin** | Display заголовки | ТОЛЬКО uppercase, крупный размер (28-48px) |
| **Decor Italic** | Декоративные подписи | Слово «выборы», мелкие акценты, italic |
| **SF Pro Display / Inter** | UI текст | Весь body-текст, кнопки, навигация |

### Правила стиля
- Без теней, без градиентов на фоне
- Бордеры: `0.5px solid #CDDCDC` (или с opacity)
- Скругления: `12-16px` для карточек, `40px` (pill) для кнопок и бейджей
- Розовый `#FFCECE` — ТОЛЬКО для бейджей статуса

## 2. Структура страницы

### 2.1 Header
**Компонент:** `Header`

- Логотип СПбГУТ (слева, 112x62px)
- Центральная эмблема студсовета (абсолютное позиционирование)
- Пилл-блок справа:
  - Кнопка переключения языка (`ru` / `en`)
  - Кнопка «войти» (фон #e7e7e7, текст #000, radius 8px)
  - Или UserDropdown (после авторизации)

**Состояния:**
- Не авторизован: кнопка «войти»
- Авторизован: дропдаун с именем, ролью, кнопкой «выйти»

**Hover:** Кнопки — `background: rgba(231, 231, 231, 0.08)`

### 2.2 Hero
**Компонент:** `Hero`

- Большой логотип выборов (зависит от языка)
- Занимает всю ширину контейнера

### 2.3 Карточка выборов (главная)
**Компонент:** `ElectionCard`

- Название выборов (Serpantin, uppercase)
- Декоративная подпись «выборы» (Decor Italic)
- Дата, время в бейджах
- Счётчик дней до выборов
- Кнопка перехода (открывает модалку)

**Бейджи:**
- `outline-pink` — бордер #FFCECE, текст #FFCECE
- `pink` — фон #FFCECE, текст #243434
- `outline` — бордер #CDDCDC, текст #CDDCDC

### 2.4 Секция архива
**Компонент:** `ArchiveSection`

- Заголовок «АРХИВ» (Serpantin, 48px, uppercase)
- Стопка карточек с 3D-перспективой (perspective: 1800px)
- Навигация по годам (кнопки внизу)
- Счётчик «1 / 3»

**Карточка архива:**
- Фон: `rgba(88, 139, 139, 0.17)`, `backdrop-filter: blur(81.15px)`
- Бордер: `0.5px solid #636363`
- Скругление: `12px`
- Содержимое: заголовок, бейджи, проценты кандидатов, явка

**Анимации:**
- Page-turn: карточка уходит вправо-вверх с rotate и fade (0.46s tween)
- Стопка: spring-анимация (stiffness 260, damping 34)
- Задние карточки: дыхание (плавное покачивание по Y)
- Раскрытие советов: spring (stiffness 300, damping 30)

**Процент кандидата (PctBlock):**
- Число: Serpantin, 60px (compact: 40px)
- Знак %: Decor Italic, 30px (compact: 20px)
- Имя: SF Pro, 13px (compact: 11px)
- Победитель: цвет `--accent`, остальные — `rgba(205, 220, 220, 0.4)`

**Явка (Turnout):**
- Скобки: Serpantin, 80px
- Число %: Serpantin, 80px
- Знак %: Decor Italic, 52px
- Подпись «явка»: SF Pro italic, 25px

### 2.5 FAQ
**Компонент:** `FAQSection`

- Аккордеон с вопросами-ответами
- Стрелка-индикатор раскрытия

### 2.6 Footer
**Компонент:** `Footer`

- Фон: `#050505`
- Информация об университете, ссылки

**Содержимое:**
- Кнопка «назад» (14px, muted)

## 4. Интерактивные состояния

### Hover
- Кнопки: `opacity: 0.85` или `background: rgba(231, 231, 231, 0.08)`
- Outline-кнопки: border-color переходит к solid accent
- Кнопки-заливки: opacity снижается
- Subtle glow: `box-shadow: 0 4px 30px 0 rgba(255, 255, 255, 0.02)` на всех интерактивных элементах
- Transition: `0.15s` для большинства, `0.2s` для крупных элементов

### Курсор
- Кастомный курсор (CursorGlow) — свечение следует за мышью
- Все элементы: `cursor: none !important`

### Фокус
- Модалки: focus trap (Tab не выходит за пределы)
- Escape закрывает модалку
- Первый focusable элемент получает фокус при открытии

## 5. Адаптивность

### Breakpoints
- **Desktop:** > 1024px
- **Tablet:** 769px – 1024px
- **Mobile:** ≤ 768px

### Tablet (≤ 1024px)
- Шрифты заголовков: -20-25%
- Стопка архива: проценты 48px
- Явка: скобки/числа 56px

### Mobile (≤ 768px)
- Layout: stack вместо row
- Padding: 20px
- Модалка: fullscreen (height 100vh, radius 0)
- Header: логотип 36px height
- Архив: стопка 540px height
- Карточка кандидата: фото/инфо в столбец, центрирование
- Gap #root: 32px
